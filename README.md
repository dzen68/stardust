# 🌌 Stardust - NASA Image Explorer

Stardust is a sleek, responsive, space-themed web application that connects directly to the NASA API to fetch and display the **Astronomy Picture of the Day (APOD)**. Built using modern frontend workflows, it allows users to explore breathtaking snapshots of our universe across different dates.

## 🚀 Features

* **Live NASA Data:** Fetches real-time imagery and detailed scientific descriptions directly from NASA's databases.
* **Smart Media Handling:** Seamlessly detects and displays standard images, direct video streams, and YouTube interactive iframes depending on the day's payload.
* **Time Travel Explorer:** Includes an interactive date picker allowing users to search historical APOD archives instantly.
* **Immersive Dark Mode CSS:** Custom UI themed with deep cosmic color palettes (`#0f172a` space slate), sleek borders, and crisp shadows optimized for rendering high-fidelity astral images.
* **Fast Build Pipelines:** Bundled and served utilizing Vite for lightning-fast hot module replacement (HMR) during development.

---

## 🛠️ Tech Stack

* **Runtime/Environment:** [Node.js](https://nodejs.org/)
* **Build Tool / Dev Server:** [Vite](https://vite.dev/)
* **Frontend Technologies:** Semantic HTML5, Custom CSS3 (Flexbox, variables, adaptive layouts), Vanilla JavaScript (ES6+, Async/Await, Fetch API)
